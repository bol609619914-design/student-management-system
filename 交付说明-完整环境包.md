# 完整环境交付包说明

本交付包用于在另一台 Windows 电脑上快速部署本项目，包内已包含：

- 项目源码
- JDK 17
- Maven 3.9.14
- MySQL Server 8.4
- 已初始化的 MySQL 数据目录
- 数据库导出文件
- 启动 / 停止 MySQL 脚本
- 中文部署说明

## 一、包内目录说明

- `project`：项目源码目录，用 IDEA 打开这里
- `runtime\jdk`：随包附带的 JDK
- `runtime\maven`：随包附带的 Maven
- `runtime\mysql`：随包附带的 MySQL Server
- `runtime\mysql-data`：已初始化的数据目录
- `scripts`：环境配置与 MySQL 启停脚本
- `sql`：数据库导出文件

## 二、推荐部署方式

### 方式一：老师电脑已安装 IDEA

1. 解压完整环境包
2. 先运行 `scripts\start-bundled-mysql.bat`
3. 再运行 `scripts\set-env.bat`
4. 打开 IDEA，选择 `Open`
5. 打开 `project\student-management-system`
6. 运行 `StudentManagementSystemApplication.java`

### 方式二：老师电脑已有自己的 MySQL

1. 直接导入 `sql\student_management_system.sql`
2. 修改项目中的数据库账号密码
3. 用 IDEA 打开 `project\student-management-system`
4. 运行启动类

## 三、默认数据库信息

- 主机：`127.0.0.1`
- 端口：`3306`
- 用户名：`root`
- 密码：`123456`
- 数据库：`student_management_system`

## 四、默认演示账号

- 管理员：`admin / admin123`
- 教师：`teacher / teacher123`
- 学生：`student / student123`

## 五、注意事项

- 如果老师电脑已经有占用 `3306` 的 MySQL，请先关闭原有服务再启动包内 MySQL。
- 如果老师电脑没有 IDEA，这个包仍然可以提供完整依赖，但源码运行仍建议通过 IDEA 打开。
- 第一次启动包内 MySQL 可能需要几秒钟，请等待脚本提示启动成功。
